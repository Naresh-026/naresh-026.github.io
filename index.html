Project Overview

This project is a web-based Parking information system, where students will be able to view available parking spaces in different places arounf the campus.

Features

  1. Aligns more or less with the ULM website theme.
  2. strongSign in / Sign Up page
    Custom Login page made using Wix member area.
  3. A map of the University showing parking locations in different places.
  4. Clear representation of curb colors and parking permission.
  5. Contact us page
  6. Link to the ULM Parking Portal where students can view, pay, and appeal for parking citations.
  7. Hosted using Wix which makes it easier by providing a link to the website.

Project Limitations

  1. parking reservation system is only introduced. It is not functional yet. Since all the students may not be using the website to use the Parking Service at first, the reservation system may create conflict with people who do not use the service. (For example, someone reserves a parking spot using the website and another person who does not use the service uses the parking spot because he has no idea that it has been reserved already)
  2. Since we do not have access to the actual campus database, we have used CMS. we have used dummy data for parking space to be later used for what and how many parking spaces are available.
  3. Because of time limitations, we have made only the demo of the solution that we are proposing. We have used parking lots for only Hemphill Hall and the library (The data shown on the website are not actual data).

Future Improvements

We will be continuing this project further improving the interactivity and functionality of the website. Some of the major improvements will include.

  • This service is intended to be a part of ULM online services. If the project idea is accepted by the university, students will be able to login using their ULM credentials (No need to sign up).
  • Functional Reservation System. Procedures and penalty systems will be discussed with the respective campus authorities.
  • The Data about the availability of parking will be collected using sensors (infrared and pressure).
  • The user interface will be improved.
